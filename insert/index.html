    Método insert - Inserindo Documentos

    O método insert da classe LebryDB é usado para inserir um documento em uma coleção. Vamos entender detalhadamente os parâmetros que você precisa fornecer ao usar este método.

    Parâmetro collection (str)

    O parâmetro collection representa o nome da coleção em que o documento será inserido. Em termos simples, você pode pensar em uma coleção como sendo semelhante a uma tabela em um banco de dados relacional.

    Por exemplo, se você estiver criando um sistema para armazenar informações de usuários, você pode ter uma coleção chamada 'users' para armazenar todos os documentos relacionados aos usuários do sistema.

    Exemplo de Uso do Parâmetro collection

    # Exemplo de uso do parâmetro 'collection'
db.insert('users', user_document)
    No exemplo acima, estamos inserindo o documento user_document na coleção 'users'.

    Parâmetro document (dict) O parâmetro document é um dicionário que representa o documento JSON a ser inserido na coleção. Um documento é uma estrutura de dados que contém informações que você deseja armazenar no banco de dados. Em outras palavras, é o conjunto de dados que você deseja gravar na coleção.

    Aqui está um exemplo de um documento que representa um usuário:

    Exemplo de um documento

    user_document = {
    'name': 'Alice',
    'age': 30,
    'email': 'alice@example.com'
}
    Neste exemplo, user_document é um documento que contém informações como o nome, a idade e o endereço de e-mail de um usuário.

    Exemplo de Uso do Parâmetro document

    Exemplo de uso do parâmetro 'document'

    db.insert('users', user_document)
    No exemplo acima, estamos inserindo o user_document na coleção 'users'.

    Conclusão O método insert permite que você insira informações organizadas (documentos) em uma coleção específica no LebryDB. Você precisa especificar o nome da coleção onde deseja inserir o documento e fornecer o documento em formato de dicionário.

    Esta abordagem simplifica o armazenamento e recuperação de dados em seu aplicativo Python, tornando-o mais organizado e fácil de gerenciar.
